# ShaderMaterial

Creates a [THREE.ShaderMaterial](https://threejs.org/docs/#api/en/materials/ShaderMaterial) rendered with custom shaders.

```html
<Box>
  <ShaderMaterial />
</Box>
```

Source : https://github.com/troisjs/trois/blob/master/src/materials/ShaderMaterial.js
